Sekilas Tentang Saya

Sebenarnya saya tidak terlalu yakin jika saya memiliki hobi, tetapi saya memiliki beberapa hal yang biasa saya lakukan ketika saya memiliki waktu luang, misalnya saja seperti menonton, bermain, dan membaca.

Biasanya saya menonton film anime. Anime yang saya sukai adalah anime dengan genre fantasy, seperti sword art online, guilty crown dan lain-lain. Selain menonton anime, saya juga suka membaca manga. Kebanyakan manga yang saya sukai bergenre fantasy dan slice of life.

Jika saya ingin bermain game, saya biasanya bermain game petualangan yang memberikan kebebasan kepada pemain untuk mengeksplorasi sepuasnya, seperti Assassin's Creed, need for speed, torchlight dan sebagainya. Saya juga menyukai game fps (first person shooting) game, misalnya Counter Strike. Saya sendiri kurang menyukai game yang online, karena biasanya game online memiliki tingkat kesulitan yang cukup tinggi. selain itu juga game online harus dimainkan dengan koneksi internet. Oleh karena itu kebanyakan dari game yang saya mainkan merupakan game offline.
